# Lesson 21 of Modern Embedded Systems Programming Course by Miro Samek

Created on: Dec 14, 2021

Author: realmer

This is Lab for lesson 21 of [Modern Embedded Systems Programming Course by Miro Samek](https://www.state-machine.com/video-course) done on STM32 Nucleo F401RE with custom led board made of Arduino breadboard:
- GND - R331 - Blue Led - D7
- GND - R331 - Red Led - D8
- GND - R331 - Yellow Led - D9
- GND - R331 - Green Led - D10

Have a fun.
